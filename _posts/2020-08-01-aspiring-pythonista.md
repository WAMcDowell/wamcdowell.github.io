*My long time interest in programming, recently python and resources I’ve discovered*

One of the skills I have dabbled and re-dabbled multiple times is programming. I have come to equate programming to wizardry. You can do magical things in the digital realm if you know the language of the craft. My interest goes back to my early years with my Atari 800 and  Commodore64. Since then I hobbied with VBA, CGI/ASP and for a good stretch of time Ruby on Rails (RoR). I learned a lot about web development thru RoR, but as my interest expanded into the realms of Data and Finance I discovered Python. Over the 3 months I’ve discovered a number of good python resources which I will list out here to be explored in depth at a later time.

- [Learn Python the Hard Way](https://learncodethehardway.org/python/): Very structured approach to learning the fundamentals of Python touching on CLI and web applications.
- [Python Crash Course](https://www.amazon.com/Python-Crash-Course-Eric-Matthes-ebook/dp/B07J4521M3/ref=tmm_kin_swatch_0?_encoding=UTF8&qid=&sr=): Introduction to Python followed by a project focused deeper dive covering desktop, data and web applications.
- [Automate the Boring Stuff with Python](https://www.amazon.com/Automate-Boring-Stuff-Python-2nd-ebook/dp/B07VSXS4NK/ref=pd_sim_351_1/139-9209155-5716411?_encoding=UTF8&pd_rd_i=B07VSXS4NK&pd_rd_r=293c4066-871f-41c8-a94f-cd78b2f2cbcf&pd_rd_w=CpJqO&pd_rd_wg=rqSPN&pf_rd_p=50fdecc9-bf07-4d35-ad07-39e73888f6b9&pf_rd_r=QP1JQP17421HWY8Z9DC6&psc=1&refRID=QP1JQP17421HWY8Z9DC6): Use case focused guide to using Python to accomplish routinized tasks, I especially liked the section on using Python with Google Sheets and Microsoft Excel.
- [Head First Python](https://www.amazon.com/Head-First-Python-Brain-Friendly-Guide-ebook/dp/B01N0GU0OC/ref=sr_1_1?dchild=1&keywords=headfirst+python&qid=1598626536&s=digital-text&sr=1-1): My favorite Python book with depth in topics and a fun style to improve learning.
- [Real Python](https://realpython.com/): Site with a very diverse set of Python articles, tutorials and podcasts. 
- [Django for Beginners](https://djangoforbeginners.com/): Thorough tutorial by [William S. Vincent](https://djangoforbeginners.com/) introducing Django web framework.
- [Django for Professionals](https://djangoforprofessionals.com/): Builds on Django for Beginners with more advanced topics such as Docker and security.
